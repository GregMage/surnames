surnames - a genealogical surname registry for XOOPS

Introduction
============
surnames is a XOOPS module which implements a surname registry, in
which researchers can register their interests in specific surnames
and identify possible similar interests in other researchers.

Installation
============
Copy surnames directory to the modules directory of your site, then
install like any other module in your site's administration area.

Notes
=====
This module has been tested in XOOPS version 2.5.9.
