Help: Surnames

Description

Surnames is a module to assist in maintaining a surname registry - a registry for genealogists to find other researches with an interest in a specific surname.

Permissions

Each group can be assigned these permissions:

  • Auto-Approve - Automatically approve submissions if set, otherwise surname submissions must be approved by an administrator before they will be seen.
  • Add/Edit/Delete for Others - If set, members of this group can edit surnames for other users. This is normally reserved for administrators.
